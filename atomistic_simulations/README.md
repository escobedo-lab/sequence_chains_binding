The initial configuration for mmD+mmT in chloroform is given in lammps format (initial_config_mmD_mmT_chloroform.data). The Hamiltonian Replica Exchange (HRE) simulations used the Prajwal et al. implementation from the following paper: Atomistic modeling approach for predicting association of photoacid generators in extreme ultraviolet polymeric photoresists (https://doi.org/10.1021/acs.chemmater.3c01750). The input script is also given here (in.polymer).

The Prajwal et al. HRE lammps implementation can be accessed from here: https://github.com/escobedo-lab/Hamiltonian-Replica-Exchange
